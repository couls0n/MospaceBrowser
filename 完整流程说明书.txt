## 系统级全局设定（System Context）

你正在开发一个名为 **XussBrowser** 的指纹浏览器（Anti-Detect Browser）。这是一个基于 Electron + Chromium 的专业级隐私工具，用于创建隔离的浏览器环境。

```typescript
    // 3. 如果提供 IP, 调用 GeoIP 逻辑匹配时区

  }
  
  private getTimezoneByIP(ip: string): string {

    // 如 1.0.0.0-1.0.0.255 -> Australia/Sydney
    // 返回 IANA 时区字符串
  }
  
  private generateFontList(os: string, seed: string): string[] {
    // 从模板字体中随机选择 50-150 个
    // 确保包含该 OS 必装字体 (如 Windows 必有 Segoe UI, Arial)
  }
  
  validateConsistency(config: FingerprintConfig): boolean {
    // 验证逻辑一致性:
    // - Win32 平台不会用 "Lucida Grande" 作为主要字体

    return true/false

```

#### 3.3 关键技术约束

1. **随机数种子**: 使用 profileId (UUID) 作为种子, 确保同一环境指纹恒定
2. **时区映射**: 内置常见 IP 段映射表 (无需外部 API 调用)
3. **WebGL Vendor**: 
   - Windows: Intel, NVIDIA, AMD
   - macOS: Intel Inc., Apple Inc.
   - Linux: Mesa, NVIDIA
4. **Canvas 噪声算法**: Perlin Noise, 仅修改 RGB 最低位 (视觉不可见)

```typescript
    // 3. 构建启动参数 (buildChromiumArgs)
    // 4. spawn 子进程

    // 6. 获取 WebSocket URL (fetchDebuggerURL)
```
